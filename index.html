Rhombus University
Peer Support Specialist Training Provider Project

This project tracks development of the Rhombus University application package to become an approved provider of the Medi-Cal Peer Support Specialist 80-hour core training in California.

Current working model: Hybrid delivery, with Rhombus University as the training provider and ECTLC participating as an instructional partner for applied skills exposure.

Project Status: In Development
Submission Goal: July Application Window
Provider Entity: Rhombus University
Instruction Model: Hybrid

Executive Overview

The core instructional and compliance package has largely been developed. Remaining work is focused on final assembly, administrative designation, and formal application submission.

Category Status
Curriculum and competency framework Substantially complete
Policies, assessments, and learner materials Substantially complete
Certificates, transcript, and schedules Draft complete
Administrative submission items Still to be finalized
Verified application fee: $300 for the training provider application, with $300 biennial renewal and $300 modification fee if needed later.
